Assets were taken from <https://github.com/samuelcust/flappy-bird-assets>.
Persistant storage by Yaya-Cout: <https://yaya-cout.github.io/Nwagyu/reference/apps/storage.html>
Thanks to this [Discord message](https://discord.com/channels/663420259851567114/1387308537860063373/1466176467204444424) for explaining how to make math.h work.
